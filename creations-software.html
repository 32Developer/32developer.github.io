SpaceGame

SpaceGame was the first fairly complex game I made. I originally made it to learn more about Greenfoot for a school project, as it was my first time using it. SpaceGame is a 3rd person Space shooter where the goal is to survive until wave 20. It may seem easy at first, but trust me, it gets really hard near the end!

KD Calculator

KD Calculator is a simple text based program which tells you how many kills you need to reach a certain KD on the Overcast Network. To run the program, go to the directory where kd_calc.py is stored in a terminal and type "python kd_calc.py".
